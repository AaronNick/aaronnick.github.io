周记｜第十七期

 2019.6.3  2123  5 分钟

#每周资讯

1、Chrome 的广告屏蔽功能将主要提供给企业级客户

对 Google 而言，如果你没有付费，那么你本质上是产品而不是服务对象。今年初，Google 在改进安全和性能的名义下提议改变 Chrome 的扩展系统，该提议被称为 Manifest V3，它将限制 WebRequest API 的功能，用 declarativeNetRequest 替代，广告屏蔽扩展如 uBlock Origin 将会受到影响，因为它们使用 WebRequest 在广告下载前屏蔽其请求。Google 的做法引发了社区的广泛批评，几个月后 Google 开发者回复了社区提出的几个问题，坚持了原有立场，声明 Chrome 的 WebRequest API 仍然能屏蔽用户不想要的内容，但该屏蔽功能将只提供给付费的企业级用户。广告是 Google 的最主要收入来源，而 Chrome 是目前浏览器市场上占统治地位的浏览器，限制 Chrome 的广告屏蔽显然有助于搜索巨人的商业利益。

2、微软应用商店允许 Win32 应用，UWP 之梦碎

微软用了多年时间试图推动开发者采用它的 Universal Windows Platform (UWP)平台，创建能在移动、桌面和主机上运行的应用，但应者寥寥，它的微软应用商店也未能像 Google 和 苹果的应用商店那样大热。微软承认了失败，它的应用商店开始允许 Win32 游戏，游戏开发者无需为 UWP 重构应用。游戏是最受欢迎的应用类型，通常也是下载量最高的应用。微软放弃 UWP 早有先兆。它的 Edge 浏览器最近切换到了 Chromium，远离 UWP 让它能支持 Windows 7、Windows 8 和 macOS。UWP 对应用的限制也让开发者在决定采用 UWP 前三思而行。微软最终倾听了开发者的声音，不再强推 UWP，这对开发者和微软商店的用户来说可能是个好消息。

#有趣数据

1、世界卫生组织预测，到2050年，抗药性微生物的全球死亡率将从目前的每年70万人增加到1000万人。届时，他们将超过癌症，心脏病和糖尿病成为导致死亡的主要原因。

2、刊登在美国国家科学院院刊上的一篇论文指出，2000 年至 2015 年间，全球抗生素消费量增长了 65%，各国的平均抗生素消费率从每天每 1000 名居民 16.4 次用药频度增加到 20.9 次，抗生素消费率中位数从每天每 1000 名居民 15.5 次增加到 19.5 次用药频度，增加了 25% 。

3、LGBT 社区的房价往往更高，有的甚至比普通社区高达 4倍，这种趋势并不是一个新现象。 当2012年美国房地产市场崩溃时，全国各地的购房者在 LGBT 居民众多的地区购买房产平均要支付29% 的溢价。 到2017年，这一比例上升到37% ，超过了非 LGBT 社区的恢复率。而其中的原因是：没有孩子，因此更有钱；思想更开放，这类地区容易为科技热点、中产阶级化提供肥沃土壤。
