软件｜一直在用的日记软件

六月份换了一家公司，由于全员使用 MacBook Pro 办公，促使我重新考虑将 Windows 转移到 Mac OS。思考了两天后，终于决定下单 MacBook Pro 2020 高配版。

◎ surface

其中，让我犹豫很久的原因主要是：

  1. 习惯了十多年的 Windows，怕难以适应新系统；
  2. Windows 上有许多优秀的独占软件；
  3. 还有一个很重要的原因：一个有情感且难迁移的软件——eDiary。

促使我最终决定下单的原因：

  1. 公司办公环境使然，虽然公司可以配电脑，但是换电脑的想法存在很久了；
  2. 使用自己的电脑，公司每个月还能补偿几百（虽少但也还能接受）；
  3. 经过了解，可以在 MacBook Pro 上使用诸如 Wine 等软件运行 Windows 软件。当然也可以安装虚拟机或者双系统，但是没必要如此大动干戈。

下面进入正题，介绍这款看似很普通的软件，却能够深入人心的原因。

#一、足够稳定

谈到稳定，很多人会以为是运行稳定，不易崩溃。但这里所要说的是服务的稳定，也就是说一个软件能够给你提供长达数十年的服务，这种稳定，可以说是一种靠谱的安全感。

这种安全感会让你不用担心软件会突然停掉，开发者会突然跑路。尤其是对于日记这类包含高度隐私的产品，稳定、可靠、安全非常之重要。我翻看了这款软件开发历程，从软件诞生，到一路维护过来的一条条日志，似乎就在看着一个孩子从诞生到成熟。(ps.更新日志放在文末)

#二、安全

互联网时代，自己的数据大多数掌握在大公司手里。免费提供产品、服务给你，然后换取你的数据，进而使用你的数据在广告投放等地方变现。若是自己书写内心想法的日记，也被用于变现，想想就令人毛骨悚然。

因此，我在选择日记产品的时候，有一个原则就是：要么不联网，要么联网同步有自己的加密措施。筛选下来，就真的没几个靠谱的了。直到遇到 eDiary，一个更新了二十年的日记软件。

初看界面，就好像上古时代的产品。没有扁平化的图标，界面几乎就是十几年前的 Windows 样式和交互。而我决定尝试它，是源于一个网友的评论：

我已经用它写了十几年日记了。

这句话莫名吸引我，也许是暗示它足够安全和稳定，得到网友的认可；也许是那种十几年默默书写日记的场景，有种难以描述的魅力。

eDiary 不联网，也不上传任何数据，你的日记都存在本地，并且使用了严格的加密机制，因此不必担心隐私的泄漏。关于同步问题，我的解决方案是将日记数据包放在 OneDrive，这样每次写完就会自动同步到云端，也不怕软件崩溃了数据丢失。

#三、高度自定义

可以自定义各种模板，方便插入到每次的新建日记里，省去不少烦恼。可以输出优雅的打印格式，设置好页码、页眉页脚等，导出来简直就是一本日记书。以后若是老了，打印出来放在书柜，傍晚在阳台晒着太阳，翻阅着以往的时光，实在是一件妙事。

#四、亲切感

日记这种东西，于我而言是一个重要的情感寄托。当我将几年的故事写在里面的时候，仿佛是在对一个老者倾述。这是一个慈祥且温柔的老者，他不会给你提建议，不会给你讲大道理，只是默默地听着。没有广告、没有推荐、没有任何的打扰，有的只是茶一般的安静。

在这三年多来，我写了 585 篇日记，将近 20 万字。里面有流水账，也有反省思考，多少悲欢离合都记录其中。可以说，这是我的另一个精神家园。每次打开，写下当天的日记，回顾下去年今日，时光如白驹过隙，感慨岁月不饶人。
