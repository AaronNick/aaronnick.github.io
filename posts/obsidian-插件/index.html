一个普通用户日常使用 Obsidian 的插件选择，主要是辅助输入，提升操作体验。对比较破坏 Markdown 语法的插件保持谨慎使用。该文档不定期更新，随着使用的深入进行增删调整。

最后更新：2021-12-10

#Obsidian 插件

#Activity History

  • 地址： https://github.com/Darakah/obsidian-activity-history
  • 功能： 展示类似 GitHub 的提交历史矩阵图，但只展示安装插件之后的活动记录。

#Advanced Tables

  • 地址： https://github.com/tgrosinger/advanced-tables-obsidian
  • 功能： 表格编辑增强，提升操作体验。

#Auto pair chinese symbol

  • 地址： https://github.com/renmu123/obsidian-auto-pair-chinese-symbol
  • 功能： 自动补齐中文的一些符号，例如 《》 「」 【】 等。

#Better Footnote

  • 地址： https://github.com/aidenlx/better-fn
  • 功能：悬停脚注的时候自动展示脚注内容，无需跳转文末。

#Better Word Count

  • 地址： https://github.com/lukeleppan/better-word-count
  • 功能： 更好的字数计数插件。

#Calendar

  • 地址： https://github.com/liamcain/obsidian-calendar-plugin
  • 功能： 展示日历视图，快速切换不同日期创建的日记。

#CodeMirror Options

  • 地址： https://github.com/nothingislost/obsidian-codemirror-options
  • 功能： 所见即所得插件。

#Customizable Sidebar

  • 地址： https://github.com/phibr0/obsidian-customizable-sidebar
  • 功能： 添加命令到侧边栏，或者隐藏侧边栏的按钮。

#Dangling links

  • 地址： https://github.com/graydon/obsidian-dangling-links
  • 功能： 通过侧边栏展示所有未创建的被引用的双链文件，以便快速创建。

#Dataview

  • 地址： https://github.com/blacksmithgu/obsidian-dataview
  • 功能： 查询、过滤、排序和提取笔记库里的笔记，比如获取某些标签的笔记等。

#Dialogue

  • 地址： https://github.com/holubj/obsidian-dialogue-plugin
  • 功能： 创建对话式的 Markdown 笔记，比如拿来记录一些话题的讨论，比较美观。

#Easy Typing

  • 地址： https://github.com/Yaozhuwa/easy-typing-obsidian
  • 功能： 编辑增强插件，自动格式化书写，例如自动在中英文之间添加空格。

#Editor Syntax Highlight

  • 地址： https://github.com/deathau/cm-editor-syntax-highlight-obsidian
  • 功能： 编辑模式下代码高亮。

#Excalidraw

  • 地址： https://github.com/zsviczian/obsidian-excalidraw-plugin
  • 功能： 在 Obsidian 绘制美观的 Excalidraw（草图），且支持其他笔记引用。

#Folder note

  • 地址： https://github.com/xpgo/obsidian-folder-note-plugin
  • 功能： 给文件夹创建同名文件并隐藏，在点击文件夹的时候展示该文件。结合 Dataview，可以加工展示该文件夹的分类内容，比如读书文件夹，通过 Dataview 在同名文件里创建在读、想读、读完等表单。

#Footnotes Shortcut

  • 地址： https://github.com/akaalias/obsidian-footnotes
  • 功能： 使用快捷键快速添加角标。

#Force note view mode by front matter

  • 地址： https://github.com/bwydoogh/obsidian-force-view-mode-of-note
  • 功能： 通过 yaml 配置单篇文档打开时是编辑或者预览状态，一般应用于主页，默认打开为预览状态。

#Graph Analysis

  • 地址： https://github.com/SkepticMystic/graph-analysis
  • 功能： 提供几种算法，计算并展示与当前文档相关系数高的文档，类似论文引用。
  • 状态： 目前已停用。因为个人笔记的引用链接做不到论文如此规范，不具备参考性。

#Hotkeys for specific files

  • 地址： https://github.com/Vinzent03/obsidian-hotkeys-for-specific-files
  • 功能： 给常用的文件设置一个快捷键，便于快速跳转。

#Kanban

  • 地址： https://github.com/mgmeyers/obsidian-kanban
  • 功能： 创建一个看板视图。

#Media Extended

  • 地址： https://github.com/aidenlx/media-extended
  • 功能： 支持在 Obsidian 打开 YouTube 视频，并支持打点做笔记。

#Media Extended BiliBili Plugin

  • 地址： https://github.com/aidenlx/mx-bili-plugin
  • 功能： 支持在 Obsidian 打开哔哩哔哩视频，并支持打点做笔记。

#Outliner

  • 地址： https://github.com/vslinko/obsidian-outliner
  • 功能： 列表编辑增强，并且使其像 Workflowy 或者 RoamResearch 一样美观。

#Recent Files

  • 地址： https://github.com/tgrosinger/recent-files-obsidian
  • 功能： 展示最近打开的文件列表。

#Remember cursor position

  • 地址： https://github.com/derwish-pro/obsidian-remember-cursor-position
  • 功能： 记住光标在文档中的最后位置，在浏览长文档或者来回切换文档时很有用。

#Reminder

  • 地址： https://github.com/uphy/obsidian-reminder
  • 功能： 管理待办事项，以及设置提醒。

#Search on Internet

  • 地址： https://github.com/HEmile/obsidian-search-on-internet
  • 功能： 在 Obsidian 内通过 Google/Wikipedia 查询选定的文本。

#Tasks

  • 地址： https://github.com/schemar/obsidian-tasks
  • 功能： 任务管理插件。

#Tidy Footnotes

  • 地址： https://github.com/charliecm/obsidian-tidy-footnotes
  • 功能： 自动整理笔记中的角标顺序，隐藏插件，需要使用命令面板调出使用。

#Various Complements

  • 地址： https://github.com/tadashi-aikawa/obsidian-various-complements-plugin
  • 功能：基于当前文档的字符，在书写时自动补充提醒（英/日语），也可以添加字典文件映射以达到辅助输入的效果。

#Word Splitting for Simplified Chinese in Edit Mode

  • 地址 ： https://github.com/aidenlx/cm-chs-patch
  • 功能： 增强 Obsidian 编辑模式下中文分词能力，使得双击选词更为精确。

#Workspaces Plus

  • 地址： https://github.com/nothingislost/obsidian-workspaces-plus
  • 功能： 工作区增强（需打开核心插件里的工作区插件）
